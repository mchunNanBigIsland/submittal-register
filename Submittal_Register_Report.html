🏠 About This Site

This Submittal Register Analysis Report provides comprehensive tracking and management of project submittals. The system automatically processes Excel data to generate real-time status reports, response tracking, and visual analytics for project stakeholders.

⏳ Key Terms & Definitions

  • NLT Date (No Later Than): The deadline by which responses must be submitted. Submittals received after this date are considered late per Spec Section 1330.
  • Response Date: The actual date when the response was uploaded to the project portal by the reviewing authority.
  • Submitted Date: The date when the contractor submitted the submittal for review.
  • Response Due Date: Calculated as 30 days from submission date for CM/Carollo response tracking.

🔍 Filtering & Search Functions

  • Search Bar: Type any submittal number, title, or description to filter results instantly.
  • Status Filters: Click status buttons (AC, AN, NCR, etc.) to show only submittals with that status.
  • Tab Navigation: Switch between Overview, Worksheets, and Status Lists for different views.
  • Collapsible Sections: Click headers to expand/collapse status sections for better organization.

📈 How to Review the Report

  1. Start with Overview: Check summary statistics and response tracking metrics.
  2. Use Worksheets Tab: Review submittals by discipline with search and filter capabilities.
  3. Check Status Lists: Focus on specific statuses (NCR, NR, AS) that require immediate attention.
  4. Monitor Response Tracking: Pay attention to overdue and approaching deadline items.
  5. Click Submittal Numbers: Open PDFs directly from the report for detailed review.

🛠 Navigation Tips

  • Red headers indicate critical statuses (NCR, NR, AS) requiring immediate attention.
  • Response tracking shows urgency levels: Green (On Time), Yellow (Due Soon), Red (Overdue).
  • Use the status legend above to understand all status codes and their meanings.
  • All data is automatically updated when the Excel file is refreshed.
